# large-screen
vue-cli4创建大屏看板rem适配方案

默认使用1920*1080

若宽高比大于1920/1080则居中显示，两侧留白，若小于1920/1080则顶部显示，底部留白，保证内容区域保持1920:1080,开发过程中就可以完全按照设计稿px开发
